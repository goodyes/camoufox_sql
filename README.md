# camoufox_sql
All Camoufox inside a T-SQL (MS SQL) database. Ready to use in your software. For data only. Comes empty.

Based on Camoufox v.135.0.1-beta.24 from March 15, 2025 which as of 2026 March 04 is still te last working version.

It ports all the Camoufox properties (fingerprints) into a SQL Database. 

Run schema.sql into your database, just change "USE 'YOUR_DATABASE_NAME'" for the name of your Microsoft SQL database name.
